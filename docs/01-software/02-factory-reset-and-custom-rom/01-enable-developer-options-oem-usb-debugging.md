# Enable developer options, OEM, USB debugging

Il modo più rapido è semplicemente andare su Impostazioni > Sistema > Avanzate > Opzioni sviluppatore e premere 7 volte su "Build number" e arriva un pop-up dicendo che le opzioni sviluppatore sono state sbloccate. Una volta fatto abilitare lo sblocco OEM, quindi sempre in Opzioni sviluppatore, attivare “Sblocco OEM”.

Moreover, go back to Settings > System > Developer options and enable USB debugging.
