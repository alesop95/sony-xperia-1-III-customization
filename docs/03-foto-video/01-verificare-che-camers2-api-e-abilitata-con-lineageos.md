# Verificare che Camers2 API è abilitata con LineageOS

> LEGACY - percorso GCam, abbandonato in favore delle app proprietarie Sony. Conservato come riferimento storico. Per il percorso attuale vedi `../RUNBOOK.md`.

Aaa

Puoi controllare facilmente così:

**Metodo rapido con app:**

Scarica da APKPure un’app gratuita chiamata **Camera2 API Probe**.

Apri l’app e guarda lo stato:

Se vedi **FULL** o **LEVEL_3**, la Camera2 API è abilitata e supportata a pieno, perfetto per GCam e RAW.

Se vedi **LEGACY** o **LIMITED**, ci possono essere limitazioni.
