# Installazione base Strato (ex-skyline)

I passaggi sono:

1. Scarica l'APK da GitHub o APKMirror (evita siti sospetti)
1. Installa l'app sul tuo Xperia
1. Crea le cartelle necessarie:
1. /Android/data/xyz.strato.strato/files/
1. Inserisci i file:
1. prod.keys e title.keys nella cartella keys/
1. I giochi nella cartella games/
1. Avvia l'app e cerca i giochi nella directory scelta

I miglioramenti sono:

1. Attiva profilo prestazioni massimo (o overclock se disponibile)
1. Disattiva il risparmio energetico
1. Usa un dissipatore esterno se emuli a lungo (l'Xperia scalda!)
1. Installa Magisk + LSPosed se vuoi tweak avanzati (come GPU Driver Mods)
1. Alcuni modder su GitHub offrono build modificate di Adreno GPU Drivers ottimizzate per l’emulazione
